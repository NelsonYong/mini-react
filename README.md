# mini-react
渐进式学习 React 核心原理及实现

本项目旨在通过源码实现和注释，帮助开发者深入理解 React 的核心机制与实现原理。

## 文档入口

本项目已集成 VitePress 文档，包含源码解析、实现思路和示例。

- 文档目录：`docs/`
- 启动本地文档预览：
  ```bash
  npx vitepress dev docs
  ```
- 访问地址：本地默认 http://localhost:5173

## 目录结构

- `packages/`：核心源码（react、react-dom、react-reconciler、shared 等）
- `demos/`：示例项目
- `docs/`：VitePress 文档
